Web monitoring lets Octolens find mentions across indexed pages on the open web. It covers **8B pages daily** and runs in real time.

Web is available during free trials and on Scale. By default, it is only turned on for your own brand keywords so we do not add too many mentions to your workspace or use up your mention limit on broad terms.

## What's included

Web includes indexed pages across the open web. By default, Octolens monitors English-language pages and only includes pages from domains with a Domain Rating of **30 or higher**.

You can change the Domain Rating threshold in [Global keyword settings](/quickstart/global-keyword-settings). The default is 30.

## Turn on Web for other keywords

To enable Web for a keyword that is not one of your own brand keywords:

1. Go to [Keyword management](https://app.octolens.com/me/keywords) in Octolens.
2. Edit the keyword.
3. Add **Web** as a platform.
4. Save your changes.

We keep Web off by default for non-brand keywords as a precaution. Broad terms can create a high volume of mentions quickly, which may use up your monthly mention limit.

## Control which domains are included

In Global keyword settings, you can control Web results across **all keywords**:

- **Allowed web domains:** only include Web mentions from these domains.
- **Excluded web domains:** exclude Web mentions from these domains.
- **Minimum Domain Rating:** only include pages from domains at or above the selected Domain Rating.

Allowed and excluded web domain settings apply to the Web source only.
